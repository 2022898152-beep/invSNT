IMPORTANT COMMANDS:

From Render Shell:
- Clear cache:      php artisan optimize:clear
- Run migrations:   php artisan migrate --force
- Create user:      php artisan tinker
- Check status:     php artisan about
- View routes:      php artisan route:list
